# E-Learning Website

## Description

The **E-Learning Website** is a web-based learning platform developed using **PHP, MySQL, JavaScript, HTML, CSS, and Bootstrap**.
The system allows students to browse courses, enroll in them, and access learning materials online. It also includes an **admin panel** for managing courses, lessons, students, and payments.

The platform is designed to provide a simple and efficient online learning environment for both students and administrators.


## Features

### Student Features

* Browse available courses
* View course details
* Student registration and login
* Enroll in courses
* Access lessons and learning materials
* Contact/feedback system

### Admin Features

* Admin dashboard
* Add, edit, and delete courses
* Manage lessons
* Manage students
* Monitor payments
* View feedback

## Installation

1. Clone the repository


git clone https://github.com/Partho130476/elearning-website.git


2. Move the project folder to your **XAMPP / WAMP / LAMP htdocs directory**

3. Start **Apache and MySQL**

4. Create a new database in **phpMyAdmin**

5. Import the provided **SQL database file** (if included)

6. Update the database credentials in:


----dbConnection.php


7. Open the project in your browser:


http://localhost/elearning-website
